# SymptoSync

Autonomous Patient Triage & Scheduling Agent — built for the **Autonomous Healthcare Hackathon** (June 13, 2026).

## Tech Stack
- Next.js 14 + TypeScript
- Vercel AI SDK (streaming chat + tool calling)
- xAI Grok (LLM triage engine)
- Tailwind CSS + shadcn/ui
- Deployed on Vercel

## Features
- Conversational symptom triage
- Autonomous specialty routing (general, dermatology, mental health, etc.)
- Emergency escalation with built‑in safety stop
- Mock appointment scheduling (ready for real API integration)
